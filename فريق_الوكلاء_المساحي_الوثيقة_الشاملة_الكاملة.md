## ملخص الفكرة الأساسية

**النموذج العام: التصنيف الهرمي (Hierarchical Classification)**

```
البيانات الخام (صور، ملفات، APIs)
    ↓
الوكيل الأول: استخراج وتصنيف أولي (Coarse Classification)
    ↓ (خريطة الفئات الأساسية: أراضي، مباني، طرق، جبال، سوائل)
الوكيل الثاني: تصنيف تفصيلي (Fine-grained Classification)
    ↓ (تفاصيل كل فئة: أنواع الأراضي، تصنيفات المباني، أنواع الطرق)
الوكيل الثالث: إرجاع جغرافي + تحليل + دعم قرار
    ↓ (خريطة موحدة + استنتاجات + تقرير)
الوكيل الرابع: تحليل هيدرولوجي متخصص (متوازي)
    ↓ (DEM، Flow، Slope، تصنيفات جبلي-مائية)
```

## دور كل وكيل (تفاصيل شاملة)

---

### **الوكيل الأول: معالج البيانات والاستخراج الأولي**

#### **المسؤوليات الشاملة:**

```
1. جمع البيانات من مصادر متعددة
   ├─ بيانات خارجية (من APIs):
   │  ├─ Google Earth Engine
   │  ├─ Sentinel Hub (صور أوروبية)
   │  ├─ USGS (وكالة المسح الجيولوجي الأمريكية)
   │  ├─ صور جوية عالية الدقة
   │  └─ صور فضائية من أقمار متعددة
   │
   ├─ صور (جوية، فضائية، خرائط):
   │  ├─ صور ملونة (RGB)
   │  ├─ صور متعددة الأطياف (Multispectral)
   │  ├─ صور رادار (SAR)
   │  ├─ صور حرارية
   │  └─ خرائط ورقية قديمة (مُرقمنة)
   │
   ├─ ملفات هندسية:
   │  ├─ ملفات CAD (تصاميم معمارية)
   │  ├─ ملفات SHP (Shapefiles - طبقات جغرافية)
   │  ├─ ملفات GeoJSON (بيانات جغرافية بصيغة JSON)
   │  └─ ملفات DXF (تصاميم هندسية)
   │
   ├─ بيانات مساحية:
   │  ├─ نقاط تحكم أرضية (GCPs - Ground Control Points)
   │  ├─ محطات GPS عالية الدقة
   │  ├─ بيانات LiDAR (ليزر ثلاثي الأبعاد)
   │  └─ سحب النقاط (Point Clouds) من الطائرات بدون طيار
   │
   └─ بيانات ارتفاع:
      ├─ DEM (نموذج الارتفاع الرقمي)
      ├─ DSM (نموذج السطح الرقمي)
      └─ DTM (نموذج التضاريس الرقمي)

2. توحيد نظام الإسقاط (Projection Unification)
   ├─ تحويل جميع الطبقات إلى نظام إسقاط موحد (EPSG)
   ├─ تحديد نظام الإسقاط المرجعي (مثل WGS84 أو إسقاط محلي)
   ├─ التحقق من توافق الإحداثيات
   └─ ضمان التوافق بين جميع المصادر المختلفة

3. الإرجاع الجغرافي الأولي (Georeferencing)
   ├─ ربط الصور الخام بالإحداثيات الحقيقية:
   │  ├─ استخدام نقاط تحكم أرضية (GCPs)
   │  ├─ استخدام بيانات الأقمار الصناعية المعروفة
   │  └─ استخدام معالجات هندسية دقيقة
   │
   ├─ تصحيح التشوهات الناتجة:
   │  ├─ تصحيح التشوهات الهندسية
   │  ├─ تصحيح تشوهات التضاريس (باستخدام DEM)
   │  └─ تصحيح تشويهات الكاميرا واللنسات
   │
   ├─ إنتاج صور متعامدة (Orthophoto):
   │  ├─ صور مصححة هندسياً بزاوية 90 درجة
   │  ├─ إزالة تأثير الميل والزاوية
   │  └─ بيانات جغرافية مرفقة (GeoTIFF)
   │
   └─ التحقق من جودة الإرجاع الجغرافي

4. التصنيف الأولي (Coarse Classification)
   ├─ استخراج الفئات الأساسية الخمس:
   │  ├─ أراضي (Lands/Open Ground)
   │  ├─ مباني (Buildings)
   │  ├─ طرق (Roads/Streets)
   │  ├─ جبال ومرتفعات (Mountains/Highlands)
   │  └─ سوائل (Water bodies/Lakes/Rivers)
   │
   ├─ استخدام تقنيات التصنيف الآلي:
   │  ├─ معالجة الصور (Image Processing)
   │  ├─ شبكات عصبية تلافيفية (CNN)
   │  ├─ مؤشرات طيفية (NDVI, NDBI, NDMI)
   │  └─ التعرف على الأنماط (Pattern Recognition)
   │
   ├─ تحويل خريطة التصنيف إلى كيانات جغرافية:
   │  ├─ تحويل البكسلات إلى مضلعات (Polygons)
   │  ├─ تحويل الخطوط الطويلة إلى خطوط جغرافية (Lines)
   │  └─ تحويل النقاط إلى نقاط جغرافية (Points)
   │
   └─ استخراج السمات الأساسية لكل كيان:
      ├─ المساحة (Area)
      ├─ المحيط (Perimeter)
      ├─ المركز الجغرافي (Centroid)
      ├─ الطول (Length)
      ├─ الاتجاه (Orientation)
      └─ شكل البكسل (Pixel Shape)

5. معالجة الفجوات والتضارب
   ├─ تحديد الحدود الناقصة أو غير الواضحة
   ├─ اقتراح فرضيات مع شرح سبب عدم التأكد:
   │  ├─ "الحد غير واضح بسبب غيوم في الصورة"
   │  ├─ "الكائن صغير جداً (< 5 بكسل)"
   │  └─ "تضارب بين مصدري بيانات"
   │
   ├─ حساب درجة الثقة في الفرضية:
   │  ├─ مشتقة من الأدلة المرئية وليس تقدير ذاتي
   │  ├─ نسبة التطابق مع المؤشرات الطيفية
   │  └─ توافق مع المناطق المحيطة
   │
   └─ الاتصال بالمراجع البشري عند الضرورة

6. الإرتباط مع تطبيق الصور الجوية (شبه جاهز)
   ├─ للرسم اليدوي وشبه الآلي
   ├─ واجهة رسم يدوي دقيق لحدود الأراضي
   ├─ تصحيح الكيانات الخاطئة
   ├─ ضمان السلامة الطوبولوجية:
   │  ├─ إغلاق المضلعات (Polygon Closure)
   │  ├─ منع التداخلات (No Overlaps)
   │  └─ ضمان عدم وجود فجوات بين الكيانات
   │
   └─ مراجعة الإسقاطات (Spatial Review & Rejection Loop)
      ├─ إمكانية التصحيح اليدوي الطفيف:
      │  ├─ إذا كانت إزاحة الإسقاط بسيطة
      │  ├─ يمكن تعديل نقاط التحكم (GCPs) محلياً
      │  └─ استكمال الرسم
      │
      └─ الرفض والبدء من جديد:
         ├─ إذا كان الإسقاط خاطئ بشكل كبير
         ├─ التواء أو إسقاط خاطئ
         └─ يوقف العمل ويعيد الإسقاط الخاطئ فقط
```

#### **المدخلات:**
- صور خام من APIs متعددة
- ملفات هندسية وبيانات مساحية من مصادر مختلفة
- بيانات ارتفاع (DEM) من مصادر عالمية أو محلية

#### **المخرجات:**
```
1. خريطة الفئات الأساسية (5 فئات فقط)
   ├─ صيغ متعددة: GeoJSON, SHP, GeoTIFF
   ├─ دقة مكانية: حسب دقة الصورة (عادة 2-5 متر)
   └─ تغطية: 100% من منطقة الدراسة

2. جدول السمات الأساسية (Attribute Table)
   ├─ معرّف فريد للكيان (ID)
   ├─ النوع (الفئة الأساسية من 5)
   ├─ المساحة / الطول / المحيط
   ├─ المركز الجغرافي (Coordinates)
   ├─ درجة الثقة الأولية (0-100%)
   ├─ ملاحظات عامة
   └─ تاريخ الاستخراج

3. ملف تقرير الاستخراج
   ├─ عدد الكيانات المستخرجة (إجمالي وحسب نوع)
   ├─ نسبة التغطية (Coverage Percentage)
   ├─ المناطق التي بها فجوات (Gap Areas)
   ├─ المناطق غير الموثوقة (Uncertain Areas)
   ├─ جودة الإسقاط الجغرافي
   └─ التوصيات للخطوة التالية

4. صور متعامدة مرجعية عالية الجودة (Orthophoto)
   ├─ صيغة: GeoTIFF مع البيانات الجغرافية المرفقة
   ├─ الدقة: تطابق دقة الصور الأصلية
   ├─ النطاقات الطيفية: RGB أو متعدد الأطياف
   └─ متاحة للمراجعة البشرية
```

### **الوكيل الثاني: محلل التصنيف التفصيلي**

#### **المسؤوليات الشاملة:**

```
1. استقبال مخرجات الوكيل الأول
   ├─ خريطة الفئات الأساسية (5 فئات)
   ├─ جدول السمات الأساسية (مساحة، محيط، موقع)
   ├─ صور متعامدة مرجعية (Orthophoto)
   ├─ تقرير الاستخراج الأولي
   └─ ملفات البيانات الجغرافية

2. تطبيق قواعد التصنيف التفصيلي لكل فئة
   
   أ) الأراضي الزراعية (تفاصيل في قسم منفصل)
   
   ب) المباني
   
   ج) الطرق
   
   د) الجبال والمرتفعات
   
   ه) السوائل

3. دمج البيانات الخارجية
   ├─ bيانات geoBoundaries (الحدود الإدارية الرسمية)
   ├─ بيانات ESA WorldCover (الغطاء الأرضي العالمي)
   ├─ معلومات تاريخية وأسماء تقليدية محلية
   ├─ بيانات من الوكيل الرابع (الهيدرولوجيا)
   └─ بيانات إحصائية ديمغرافية (إن وجدت)

4. حساب درجات الثقة
   ├─ ثقة كل معيار على حده
   ├─ ثقة كل نوع من الأنواع
   ├─ ثقة الكيان الكلية
   ├─ توثيق السبب وراء كل درجة ثقة
   └─ معادلات حسابية دقيقة

5. حل التعارضات والتناقضات
   ├─ التعارض بين معايير مختلفة
   ├─ التعارض مع البيانات الخارجية
   ├─ التعارض مع البيانات التاريخية
   ├─ تطبيق قاعدة ترجيح واضحة
   └─ توثيق كل قرار

6. إنتاج المخرجات
```

#### **المدخلات:**
```
من الوكيل الأول:
├─ خريطة الفئات الأساسية
├─ جدول السمات (مساحة، شكل، محيط، موقع)
├─ صور متعامدة مرجعية
└─ تقرير الاستخراج

من الوكيل الرابع:
├─ خرائط الجريان (Flow Direction)
├─ بيانات الانحدار (Slope)
├─ بيانات الارتفاع (DEM)
└─ خرائط الأودية والسيول

من مصادر خارجية:
├─ geoBoundaries
├─ ESA WorldCover
├─ بيانات تاريخية
└─ معلومات محلية
```

#### **المخرجات:**
```
1. خريطة التصنيف التفصيلي الشاملة
   ├─ لكل كيان: النوع التفصيلي المحدد
   ├─ صيغ: GeoJSON, SHP, GeoTIFF
   ├─ الألوان: مختلفة لكل نوع
   └─ درجات ثقة موضحة

2. جدول التفاصيل الشامل (Detailed Attribute Table)
   ├─ معرّف الكيان
   ├─ التصنيف الأولي (من الوكيل الأول)
   ├─ التصنيف التفصيلي (النوع المحدد)
   ├─ السمات الهندسية (مساحة، شكل، موقع)
   ├─ الخصائص الموروثة (من البيانات الخارجية)
   ├─ درجة الثقة لكل معيار
   ├─ درجة الثقة الكلية
   ├─ القواعد المطبقة في التصنيف
   ├─ حالة الكيان (موثق / غير موثق / بحاجة مراجعة)
   └─ ملاحظات تفصيلية

3. ملف تقرير التصنيف التفصيلي
   ├─ إحصائيات عامة:
   │  ├─ إجمالي الكيانات المصنفة
   │  ├─ عدد الكيانات لكل نوع
   │  ├─ النسب المئوية
   │  └─ المساحات الإجمالية
   │
   ├─ إحصائيات الثقة:
   │  ├─ متوسط درجات الثقة
   │  ├─ توزيع درجات الثقة
   │  └─ عدد الكيانات ذات ثقة منخفضة
   │
   ├─ التضاربات والتناقضات:
   │  ├─ عدد الحالات المتضاربة
   │  ├─ الأسباب الرئيسية
   │  └─ الحلول المطبقة
   │
   ├─ الحالات التي تحتاج مراجعة بشرية:
   │  ├─ القائمة التفصيلية
   │  ├─ أسباب عدم اليقين
   │  └─ المواقع الجغرافية
   │
   └─ التوصيات للتحسين:
      ├─ قواعس قد تحتاج تعديل
      ├─ بيانات خارجية بحاجة توثيق أفضل
      └─ خطوات لتحسين الدقة
```

### **الوكيل الثالث: محلل الإرجاع الجغرافي والتحليل ودعم القرار**

#### **المسؤوليات الشاملة:**

```
1. الإرجاع الجغرافي المتقدم والدقيق
   ├─ توحيد جميع الطبقات في نظام إسقاط واحد (EPSG)
   ├─ تصحيح التشوهات الهندسية المتبقية:
   │  ├─ استخدام DEM من الوكيل الرابع
   │  ├─ تصحيح تشويهات التضاريس
   │  └─ تصحيح الأخطاء الهندسية الدقيقة
   │
   ├─ إنتاج صور متعامدة نهائية عالية الجودة:
   │  ├─ صيغة: GeoTIFF مع البيانات الجغرافية
   │  ├─ دقة: عالية جداً
   │  └─ خالية من التشويهات
   │
   ├─ القص حسب منطقة الدراسة (AOI):
   │  └─ التركيز على المنطقة المهمة فقط
   │
   ├─ التحقق من جودة الإسقاط:
   │  ├─ عدم وجود فجوات بين الطبقات
   │  ├─ عدم وجود تداخلات (Overlaps)
   │  ├─ توافق الحدود بين الكيانات
   │  └─ دقة الإسقاط (± X متر)
   │
   └─ تصحيح الأخطاء الهندسية:
      ├─ حل التقاطعات (Intersections)
      ├─ ملء الفجوات الصغيرة
      ├─ تصحيح التداخلات
      └─ ضمان صحة البيانات الطوبولوجية

2. التحليل المكاني والاستنتاج المتقدم
   
   أ) كشف التغيرات عبر الزمن (Change Detection)
      ├─ المقارنة بين صور من فترات زمنية مختلفة:
      │  ├─ 2000 vs 2010
      │  ├─ 2010 vs 2020
      │  └─ 2020 vs الحالي
      │
      ├─ تحديد الكيانات التي:
      │  ├─ اختفت (حول أحمر)
      │  ├─ ظهرت جديدة (حول أزرق)
      │  └─ تغيرت خصائصها (حول أصفر)
      │
      ├─ قياس معدل التغير:
      │  ├─ متر مربع / سنة
      │  ├─ نسبة مئوية
      │  └─ سرعة التغير (سريع / متوسط / بطيء)
      │
      ├─ إنتاج خرائط:
      │  ├─ خريطة التغير (Change Map)
      │  ├─ خريطة السرعة (Velocity Map)
      │  └─ خريطة الاتجاه (Direction Map)
      │
      └─ الإحصائيات:
         ├─ إجمالي المساحة المتغيرة
         ├─ متوسط معدل التغير السنوي
         └─ الاتجاهات العامة

   ب) تحليل الأنماط المكانية (Spatial Pattern Analysis)
      ├─ تجميع (Clustering):
      │  ├─ كيف تتجمع ظاهرة معينة
      │  ├─ مراكز التجمع (Hot Spots)
      │  └─ درجة التجمع
      │
      ├─ انتشار (Dispersion):
      │  ├─ هل الظاهرة منتشرة أم مركزة
      │  ├─ مؤشرات التوزيع المكاني
      │  └─ الفراغات بين التجمعات
      │
      ├─ كثافة (Density):
      │  ├─ كثافة الكيانات في المنطقة
      │  ├─ خريطة الكثافة (Density Map)
      │  └─ مراكز التركيز العالي
      │
      └─ ارتباط (Correlation):
         ├─ العلاقات المكانية بين الظواهر
         ├─ التأثير المتبادل
         └─ الأنماط المشتركة

   ج) التفسير المتقدم مع التوثيق الكامل
      ├─ تحديد الظواهر الخطرة:
      │  ├─ مناطق معرضة للسيول (من بيانات DEM والجريان)
      │  ├─ مناطق الانجراف والتعرية (من الانحدار)
      │  ├─ مناطق عدم الاستقرار الجيولوجي
      │  └─ مناطق التلوث المحتملة
      │
      ├─ تقدير الحالة والتطور:
      │  ├─ مناطق التوسع العمراني السريع
      │  ├─ مناطق النمو الزراعي
      │  ├─ مناطق التدهور البيئي
      │  └─ مناطق التطور الاقتصادي
      │
      └─ التفسير مع التوثيق:
         ├─ توثيق الأدلة المستخدمة
         ├─ درجة الثقة في التفسير
         ├─ الفرضيات البديلة
         └─ محاذير التفسير

3. حل التعارضات والتناقضات الشاملة
   ├─ عند تضارب نتائج من مصادر مختلفة:
   │  ├─ قارن بيانات الوكيل الثاني مع البيانات الخارجية
   │  ├─ احسب درجة الاتفاق (Agreement Score)
   │  ├─ في حالة الاختلاف: طبّق قاعدة الأولوية
   │  │  ├─ 1️⃣ البيانات الأحدث (الصور الأخيرة)
   │  │  ├─ 2️⃣ البيانات الموثوقة أكثر
   │  │  │  ├─ مصادر حكومية > عامة
   │  │  │  └─ صور عالية الدقة > منخفضة الدقة
   │  │  ├─ 3️⃣ البيانات المرجعية الرسمية
   │  │  └─ 4️⃣ التحكيم البشري
   │  │
   │  └─ وثّق القرار والسبب في تقرير خاص

4. دعم القرار الشامل
   ├─ صياغة التقرير النهائي:
   │  ├─ ملخص تنفيذي (صفحة واحدة)
   │  ├─ الخرائط والمخططات الملونة
   │  ├─ جداول الإحصائيات التفصيلية
   │  ├─ الاستنتاجات والتفسيرات
   │  ├─ التوصيات المرتبة حسب الأولوية
   │  ├─ البدائل والسيناريوهات المحتملة
   │  └─ خطط العمل المقترحة
   │
   ├─ عرض التقرير على المراجع البشري:
   │  ├─ مناقشة النتائج
   │  ├─ التحقق من الصحة
   │  ├─ طلب التعديلات إذا لزمت
   │  └─ التصديق النهائي
   │
   └─ توثيق عملية الموافقة:
      ├─ توقيع المراجع
      ├─ التاريخ والوقت
      ├─ أي تعليقات أو تحفظات
      └─ النسخة النهائية المعتمدة
```

#### **المدخلات:**
```
من الوكيل الثاني:
├─ خريطة التصنيف التفصيلي
├─ جدول التفاصيل الشامل
└─ تقرير التصنيف

من الوكيل الرابع:
├─ بيانات DEM
├─ خرائط الجريان والانحدار
├─ خرائط الأودية والسيول
└─ تقييمات الأخطار المائية

مصادر خارجية:
├─ صور تاريخية قديمة (للمقارنة)
├─ بيانات حدود إدارية موثوقة
├─ خرائط رسمية معتمدة
└─ بيانات احصائية ديمغرافية

من المستخدم/الخبير:
├─ معايير قرار مخصصة
├─ أولويات التحليل
└─ أسئلة محددة يجب الإجابة عليها
```

#### **المخرجات:**
```
1. خريطة موحدة نهائية عالية الدقة
   ├─ كل كيان محدد بدقة جغرافية عالية (± 2-5 متر)
   ├─ تصنيف تفصيلي موثوق (> 80% دقة)
   ├─ درجة ثقة شاملة موثقة
   └─ صيغ: GeoTIFF, GeoJSON, SHP

2. خرائط تحليلية متخصصة:
   ├─ خرائط التغير عبر الزمن (مع أسهم اتجاه)
   ├─ خرائط كثافة التوزيع (بتدرج لوني)
   ├─ خرائط الأخطار والتعريضات
   ├─ خرائط الإمكانيات والموارد
   ├─ خرائط الاتجاهات والسيناريوهات المستقبلية
   └─ خرائط الأولويات (حسب معايير القرار)

3. جداول إحصائية شاملة:
   ├─ إحصائيات الكيانات:
   │  ├─ العدد الكلي لكل نوع
   │  ├─ المساحات الإجمالية
   │  ├─ متوسط الأحجام
   │  └─ النسب المئوية
   │
   ├─ مقاييس التغير:
   │  ├─ المساحة المتغيرة (بالمتر المربع والنسبة)
   │  ├─ معدلات النمو والتراجع (سنوياً)
   │  ├─ المناطق الأسرع تغييراً
   │  └─ الاتجاهات العامة
   │
   ├─ معدلات الاستدامة:
   │  ├─ مؤشرات البيئة
   │  ├─ مؤشرات الموارد
   │  └─ مؤشرات التطور
   │
   └─ الأولويات والفرص:
      ├─ المناطق الحرجة (Priority Areas)
      ├─ مناطق التدخل المقترحة
      └─ الموارد المتاحة

4. تقرير دعم القرار الشامل:
   ├─ الجزء 1: الملخص التنفيذي
   │  ├─ الوضع الحالي (صفحة واحدة)
   │  ├─ النتائج الرئيسية
   │  └─ التوصيات الأساسية
   │
   ├─ الجزء 2: التحليل التفصيلي
   │  ├─ الوضع الحالي مفصلاً
   │  ├─ التغيرات والاتجاهات
   │  ├─ الأخطار والفرص
   │  └─ العوامل المؤثرة
   │
   ├─ الجزء 3: الخرائط والمخططات
   │  ├─ خرائط ملونة عالية الجودة
   │  ├─ مخططات توضيحية
   │  ├─ رسوم بيانية
   │  └─ صور مقارنة
   │
   ├─ الجزء 4: الجداول والإحصائيات
   │  ├─ جداول شاملة
   │  ├─ إحصائيات مفصلة
   │  └─ مؤشرات أداء
   │
   ├─ الجزء 5: الاستنتاجات والتفسيرات
   │  ├─ ما تعنيه النتائج
   │  ├─ الآثار المحتملة
   │  └─ السيناريوهات المستقبلية
   │
   ├─ الجزء 6: التوصيات
   │  ├─ التوصيات مرتبة حسب الأولوية
   │  ├─ تقدير الجدوى والتكلفة
   │  └─ المتطلبات لتطبيق كل توصية
   │
   └─ الجزء 7: خطط العمل
      ├─ مراحل التنفيذ
      ├─ الجهات المسؤولة
      ├─ الموارد المطلوبة
      └─ المؤشرات الزمنية
```

### **الوكيل الرابع: محلل البيانات الهيدرولوجية والتضاريسية**
**(يعمل بالتوازي مع الوكلاء الأخرى)**

#### **المسؤوليات الشاملة:**

```
1. استخراج وتحليل بيانات الارتفاع (DEM)
   ├─ الحصول على بيانات DEM من مصادر متعددة:
   │  ├─ SRTM (Shuttle Radar Topography Mission)
   │  │  ├─ دقة: 30 متر
   │  │  ├─ تغطية: 90% من الأرض
   │  │  └─ المصدر: وكالة ناسا والمسح الجيولوجي الأمريكي
   │  │
   │  ├─ ALOS PALSAR (Advanced Land Observing Satellite)
   │  │  ├─ دقة: 12.5 متر
   │  │  ├─ تغطية: عالمية
   │  │  └─ المصدر: الوكالة اليابانية لاستكشاف الفضاء
   │  │
   │  └─ مصادر محلية عالية الدقة:
   │     ├─ LiDAR (Light Detection and Ranging)
   │     │  └─ دقة: 0.5-2 متر (عالية جداً)
   │     │
   │     └─ صور من طائرات بدون طيار (UAV/Drones)
   │        └─ دقة: 1-10 سم (عالية جداً)
   │
   ├─ معالجة بيانات DEM الخام:
   │  ├─ ملء الفراغات (Gap Filling)
   │  ├─ تصحيح الأخطاء (Error Correction)
   │  ├─ تنعيم البيانات (Smoothing) إذا لزم
   │  └─ التحقق من الجودة (Quality Control)
   │
   └─ إنتاج DEM محسّن وموثوق

2. تحليل خصائص التضاريس بالتفصيل
   
   أ) خريطة الانحدار (Slope Map)
      ├─ حساب درجة الميل لكل نقطة
      ├─ تصنيف الانحدار:
      │  ├─ مسطح جداً: 0-2°
      │  ├─ مسطح: 2-5°
      │  ├─ متموج: 5-15°
      │  ├─ مرتفع: 15-30°
      │  ├─ شديد: 30-45°
      │  └─ شديد جداً: > 45°
      │
      ├─ الفوائد:
      │  ├─ تحديد المناطق الصالحة للزراعة
      │  ├─ تقدير مخاطر الانجراف والتعرية
      │  ├─ تقدير توفر المياه (الرطوبة)
      │  └─ التخطيط العمراني والطرق
      │
      └─ المخرجات:
         ├─ خريطة الانحدار بألوان مختلفة
         └─ جداول إحصائية

   ب) خريطة الاتجاه (Aspect Map)
      ├─ حساب اتجاه المنحدر:
      │  └─ الاتجاهات الثمانية: N, NE, E, SE, S, SW, W, NW
      │
      ├─ الفوائد:
      │  ├─ تحديد كمية الإشعاع الشمسي
      │  ├─ تقدير النمو النباتي
      │  ├─ تحديد المناطق الحارة والباردة
      │  └─ التنبؤ بالظروف الرطبة والجافة
      │
      └─ المخرجات:
         ├─ خريطة الاتجاه
         └─ إحصائيات التوزيع

   ج) خريطة التحدب (Curvature Map)
      ├─ حساب تحدب المنحدر:
      │  ├─ محدب (Convex): تجميع الجريان
      │  ├─ مسطح (Flat): محايد
      │  └─ مقعر (Concave): تركيز الجريان
      │
      ├─ الفوائد:
      │  ├─ توقع تركز مياه الجريان
      │  ├─ تحديد خطوط التصريف الطبيعية
      │  └─ تقدير معدلات التعرية
      │
      └─ المخرجات:
         ├─ خريطة التحدب
         └─ جداول الخصائص

   د) استخراج خطوط الكنتور (Contour Lines)
      ├─ خطوط الارتفاع المتساوي:
      │  ├─ فترات 10 متر (تفاصيل عالية)
      │  ├─ فترات 20 متر (معيارية)
      │  └─ فترات 50 متر (نظرة عامة)
      │
      ├─ الفوائد:
      │  ├─ تصور واضح للتضاريس
      │  ├─ حساب الحجوم والمسافات
      │  └─ تخطيط المشاريع الهندسية
      │
      └─ المخرجات:
         ├─ خرائط الكنتور
         └─ ملفات CAD للمهندسين

   ه) تحديد قمم الجبال والأكمات والوديان
      ├─ الخوارزمية:
      │  ├─ البحث عن نقاط الحد الأقصى (Peaks)
      │  ├─ البحث عن نقاط الحد الأدنى (Valleys)
      │  └─ حساب الارتفاعات بدقة
      │
      ├─ الفوائد:
      │  ├─ تحديد المعالم الجغرافية الرئيسية
      │  ├─ إسناد الأسماء الجغرافية
      │  └─ التخطيط والملاحة
      │
      └─ المخرجات:
         ├─ قوائم قمم الجبال (مع الارتفاعات)
         └─ خريطة المعالم

3. التحليل الهيدرولوجي الشامل
   
   أ) حساب اتجاه الجريان (Flow Direction)
      ├─ خوارزمية D8:
      │  ├─ تحديد اتجاه الجريان من كل خلية
      │  ├─ الاتجاه نحو أقرب جار منخفض
      │  └─ 8 اتجاهات ممكنة
      │
      ├─ النتيجة:
      │  ├─ خريطة اتجاه الجريان
      │  └─ أسهم توضح اتجاه المياه
      │
      └─ الفوائد:
         ├─ تتبع مسار المياه
         ├─ تحديد مناطق التجميع
         └─ التنبؤ بمسارات السيول

   ب) استخراج شبكة الأودية والسيول الكاملة
      ├─ تحديد مجاري الأودية الرئيسية:
      │  ├─ الوادي الأم (Main Wadi)
      │  ├─ الروافد الكبرى (Major Tributaries)
      │  └─ الروافد الصغرى (Minor Tributaries)
      │
      ├─ استخراج الجداول والقنوات الصغيرة:
      │  ├─ تصريفات موسمية
      │  └─ قنوات صغيرة دائمة
      │
      ├─ ترتيب الأودية (Strahler Order):
      │  ├─ المستوى الأول: الينابيع الصغيرة
      │  ├─ المستوى الثاني: تجميع جديول (2 من المستوى الأول)
      │  └─ الترتيب يزداد مع كل تجميع
      │
      ├─ حساب كثافة التصريف:
      │  ├─ طول الأودية / مساحة الحوض
      │  ├─ مؤشر للنشاط الهيدرولوجي
      │  └─ عالي = منطقة نشطة مائياً
      │
      └─ المخرجات:
         ├─ خريطة شبكة الأودية والسيول
         ├─ جداول خصائص كل وادٍ
         └─ إحصائيات الكثافة والرتبة

   ج) استخراج المستجمعات المائية (Watersheds/Basins)
      ├─ تحديد حدود كل حوض مائي:
      │  ├─ كل خلية تتدفق نحو مخرج مشترك
      │  ├─ حدود واضحة بين الأحواض
      │  └─ لا توجد تداخلات
      │
      ├─ حساب خصائص كل حوض:
      │  ├─ مساحة الحوض (كم²)
      │  ├─ محيط الحوض
      │  ├─ طول أطول وادٍ (Main Channel Length)
      │  ├─ عدد الروافد
      │  ├─ متوسط الانحدار
      │  ├─ الارتفاع الأقصى والأدنى
      │  └─ معامل الشكل (Form Factor)
      │
      ├─ تحديد مخارج الأودية (Outlets):
      │  ├─ نقاط خروج المياه من الحوض
      │  ├─ أنهار رئيسية أو بحار
      │  └─ ترابطها بشبكة الأودية
      │
      └─ المخرجات:
         ├─ خريطة المستجمعات المائية
         ├─ ملفات شكاً لكل حوض
         └─ جداول الخصائص المفصلة

   د) استخراج المسطحات المائية
      ├─ البحيرات والمستنقعات:
      │  ├─ تحديد حدود دقيقة
      │  ├─ حساب المساحة والعمق
      │  └─ تصنيف حسب الحجم
      │
      ├─ مصادر المياه:
      │  ├─ العيون المائية (Springs)
      │  ├─ الآبار
      │  ├─ السدود والخزانات
      │  └─ الينابيع الحارة
      │
      ├─ مسارات المياه الدائمة:
      │  ├─ أنهار دائمة الجريان
      │  ├─ خطوط ثابتة على الخريطة
      │  └─ الكثافة والمدى
      │
      ├─ الخطوط الموسمية:
      │  ├─ تظهر فقط في موسم الأمطار
      │  ├─ تتلاشى في الفصول الجافة
      │  └─ محددة بخطوط متقطعة
      │
      └─ المخرجات:
         ├─ خريطة المسطحات المائية
         ├─ جداول المواقع والمساحات
         └─ معلومات عن كل مصدر مائي

   ه) تقدير الحطر المائي
      ├─ مناطق معرضة للفيضانات:
      │  ├─ منطقة التدفق من السد العالي
      │  ├─ أراضي منخفضة بجانب الأودية
      │  ├─ السهول الفيضية (Floodplains)
      │  └─ درجات الخطر (منخفض، متوسط، عالي)
      │
      ├─ مناطق معرضة للسيول:
      │  ├─ مناطق على امتداد وادي مع ميول عالية
      │  ├─ مناطق التجميع السريع
      │  └─ درجات الخطر
      │
      ├─ معدلات الجريان السطحي:
      │  ├─ كمية المياه المتوقع تدفقها (م³/ث)
      │  ├─ معادلات رياضية دقيقة
      │  └─ السيناريوهات المختلفة
      │
      └─ فترات العودة (Return Periods):
         ├─ فيضان 10 سنوات
         ├─ فيضان 50 سنة
         ├─ فيضان 100 سنة
         └─ فيضان نادر جداً (>100 سنة)

4. تصنيف الجبال والمرتفعات والسوائل
   
   أ) تصنيف الجبال حسب الارتفاع
      ├─ جبال عالية جداً: > 2500 متر
      ├─ جبال عالية: 2000-2500 متر
      ├─ جبال متوسطة: 1000-2000 متر
      ├─ تلال: 500-1000 متر
      ├─ أكمات صغيرة: 200-500 متر
      └─ ارتفاعات منخفضة: < 200 متر

   ب) خصائص الجبال
      ├─ تصنيف الصخور:
      │  ├─ صخور نارية (Igneous)
      │  ├─ صخور رسوبية (Sedimentary)
      │  ├─ صخور متحولة (Metamorphic)
      │  └─ مخاليط (Mixed)
      │
      ├─ درجة التعرية:
      │  ├─ بسيطة: حد أدنى من التعرية
      │  ├─ متوسطة: تعرية ملحوظة
      │  ├─ عالية: تعرية كبيرة
      │  └─ شديدة جداً: تعرية مكثفة
      │
      ├─ الغطاء النباتي:
      │  ├─ غابات كثيفة: > 70% غطاء
      │  ├─ غابات متوسطة: 30-70% غطاء
      │  ├─ نباتات شحيحة: < 30% غطاء
      │  └─ عارية تماماً: 0% غطاء
      │
      └─ الخطور الجيولوجية:
         ├─ انهيارات أرضية (Landslides)
         ├─ انزلاقات صخرية (Rock Falls)
         ├─ عدم الاستقرار الزلزالي
         └─ تصنيف المخاطر (منخفض-عالي)

   ج) تصنيف السوائل
      ├─ الأنهار والجداول:
      │  ├─ أنهار دائمة: تجري طول السنة
      │  ├─ أنهار موسمية: تجري في موسم المطر فقط
      │  ├─ جداول صغيرة: دائمة أو موسمية
      │  └─ وديان جافة: تملأ عند السيول فقط
      │
      ├─ البحيرات والمستنقعات:
      │  ├─ بحيرات دائمة: مياه عذبة
      │  ├─ بحيرات موسمية: تختفي في الصيف
      │  ├─ مستنقعات: برك صغيرة بنباتات
      │  └─ مناطق معرقة: رطبة بدون ماء سطحي
      │
      ├─ المسطحات المالحة:
      │  ├─ بحار وخلجان
      │  ├─ محيطات
      │  ├─ بحيرات مالحة
      │  └─ مستنقعات مالحة
      │
      └─ نوعية المياه:
         ├─ مياه عذبة (Freshwater)
         ├─ مياه قلوية (Alkaline)
         ├─ مياه كبريتية (Sulfurous)
         └─ مياه مالحة (Saline)

5. إنتاج خرائط مركبة متخصصة
   ├─ خريطة المخاطر الهيدرولوجية:
   │  ├─ مناطق فيضان (أحمر)
   │  ├─ مناطق سيول (برتقالي)
   │  ├─ مناطق آمنة (أخضر)
   │  └─ مؤشرات رقمية للشدة
   │
   ├─ خريطة الملائمة (Suitability):
   │  ├─ ملائمة للزراعة (درجات من 0-100)
   │  ├─ ملائمة للعمران
   │  ├─ ملائمة للسياحة
   │  └─ ملائمة للصناعة
   │
   ├─ خريطة قدرات التحمل (Carrying Capacity):
   │  ├─ كم شخص/هكتار يمكن أن تدعمه الأرض
   │  ├─ كم حيوان/هكتار للرعي
   │  └─ كم محصول/هكتار للزراعة
   │
   └─ خريطة التوصيات للاستخدام الأمثل:
      ├─ زراعة مكثفة: مناطق قابلة
      ├─ زراعة محدودة: مناطق حدودية
      ├─ محميات طبيعية: مناطق حساسة
      └─ عدم التطوير: مناطق خطرة
```

#### **المدخلات:**
```
- بيانات ارتفاع DEM (من SRTM, ALOS, أو مصادر محلية)
- صور جوية/فضائية (لتحديد المسطحات المائية)
- بيانات هطول الأمطار (اختياري - لتحسين الدقة)
- بيانات تدفق المياه التاريخية (اختياري - للتحقق)
- خرائط جيولوجية قديمة (اختياري - للمقارنة)
```

#### **المخرجات:**
```
1. نماذج جغرافية رقمية:
   ├─ DEM محسّن (10-30 متر دقة)
   ├─ خرائط انحدار (Slope)
   ├─ خرائط اتجاه (Aspect)
   ├─ خرائط كنتور (5 مستويات من التفاصيل)
   └─ خرائط تحدب (Curvature)

2. خرائط هيدرولوجية شاملة:
   ├─ خريطة الجريان (Flow Direction)
   ├─ خريطة تجميع الجريان (Flow Accumulation)
   ├─ خريطة الأودية والسيول (شاملة)
   ├─ خريطة المستجمعات المائية (مع الحدود الدقيقة)
   ├─ خريطة المسطحات المائية (نقاطية وخطية)
   └─ خريطة المخاطر المائية (ملونة)

3. جداول إحصائية شاملة:
   ├─ إحصائيات الارتفاع:
   │  ├─ min, max, mean, median
   │  ├─ الانحراف المعياري
   │  └─ التوزيع التكراري
   │
   ├─ إحصائيات الانحدار:
   │  ├─ درجات مختلفة (بالنسب المئوية)
   │  └─ المناطق الصالحة والخطرة
   │
   ├─ خصائص الأحواض المائية:
   │  ├─ المساحة والمحيط
   │  ├─ الطول والعرض والشكل
   │  ├─ معدل الانحدار
   │  └─ كثافة التصريف
   │
   ├─ خصائص الأودية:
   │  ├─ الطول والعرض
   │  ├─ معدل الجريان
   │  └─ الرتبة والترتيب
   │
   └─ خصائص المسطحات المائية:
      ├─ الموقع (إحداثيات)
      ├─ النوع (نهر، بحيرة، عين، إلخ)
      ├─ المساحة / الطول
      └─ نوعية المياه

4. تقارير تحليلية شاملة:
   ├─ ملخص الخصائص الهيدرولوجية
   ├─ تقييم مفصل للأخطار المائية
   │  ├─ مناطق الفيضان المحتملة
   │  ├─ سرعات السيول المتوقعة
   │  └─ فترات العودة
   │
   ├─ تقييم موارد المياه:
   │  ├─ كميات المياه المتاحة
   │  ├─ معدلات التدفق
   │  └─ إمكانيات الاستخدام
   │
   ├─ التوصيات للإدارة المائية:
   │  ├─ إنشاء سدود وخزانات
   │  ├─ حماية من الفيضانات
   │  ├─ استخدام مستدام للمياه
   │  └─ حماية الأنهار والعيون
   │
   └─ السيناريوهات المستقبلية:
      ├─ تأثير التغير المناخي
      ├─ تأثير الاستخدام البشري
      └─ إجراءات التكيف المقترحة
```

| 6 | **جِرْبَة** | 2000-5000 م² | > 8 م | طويل | شكل منتظم | كبير، يستغرق حرثه 2-3 ساعات | عالية |

#### **معايير القرار الخطوة بخطوة:**

```
الخطوة 1: احسب المساحة من خريطة الوكيل الأول
  └─ استخدم الصيغة: المساحة = عدد البكسلات × حجم البكسل²

الخطوة 2: احسب الأبعاد الهندسية من شكل الكيان
  ├─ العرض: أقصر مسافة عمودية على الطول
  ├─ الطول: أطول مسافة في الشكل
  └─ المحيط: محيط الشكل الكامل

الخطوة 3: ابحث في الجدول أعلاه عن التطابق
  ├─ ابدأ بالمساحة كمعيار أساسي
  ├─ ثم تحقق من العرض والطول
  ├─ تحقق من المعايير الإضافية
  └─ حسّن النوع المرشح

الخطوة 4: احسب نسبة التطابق
  ├─ هل تطابق المساحة: كم نسبة التطابق؟
  ├─ هل تطابق العرض: كم نسبة التطابق؟
  ├─ هل تطابق الطول: كم نسبة التطابق؟
  └─ متوسط النسب = نسبة التطابق الكلية

الخطوة 5: اتخذ القرار بناءً على النسبة:
  ├─ إذا كان التطابق > 85% → اعتمد التصنيف (ثقة عالية جداً)
  ├─ إذا كان 70-85% → اعتمد التصنيف (ثقة عالية)
  ├─ إذا كان 50-70% → اعتمد مع ملاحظة (ثقة متوسطة)
  ├─ إذا كان 30-50% → علّم للمراجعة البشرية (ثقة منخفضة)
  └─ إذا كان < 30% → رفض وابدأ من جديد (لا تطابق)

الخطوة 6: وثّق القرار:
  ├─ النوع المختار
  ├─ نسبة التطابق
  ├─ المعايير المطابقة والمخالفة
  ├─ درجة الثقة النهائية
  └─ أي ملاحظات خاصة
```

#### **المحور الثاني: موقع الأرض بالنسبة للوادي والتضاريس**

| النوع | الموقع | الدلالة | المؤشرات من DEM | الأهمية |
|--------|--------|----------|------------------|----------|
| **وادي** | وسط منطقة منخفضة | محاذية لمجاري السيول | High Flow Accumulation | عالية |
| **شعب** | بين مرتفعين | محصورة بين حيود | Valley between Ridges | عالية |
| **ظهرة** | على مرتفع | على هضبة صغيرة | High Elevation Ridge | متوسطة |

**كيفية التحديد:**
```
من خريطة DEM من الوكيل الرابع:
├─ احسب Elevation لمنطقة الأرض
├─ احسب تجميع الجريان (Flow Accumulation)
├─ احسب الانحدار (Slope) المحيط

الآن قرّر:
├─ إذا كانت:
│  ├─ Flow Accumulation عالي جداً (> 1000)
│  ├─ في منطقة منخفضة نسبياً
│  └─ بجانب نهر أو وادي → "وادي"
│
├─ إذا كانت:
│  ├─ محصورة بين مرتفعات (Ridges)
│  ├─ Flow Direction نحو مركز ضيق
│  └─ شكل طولي في اتجاه الجريان → "شعب"
│
└─ إذا كانت:
   ├─ على ارتفاع عالي
   ├─ على هضبة (Plateau)
   └─ لا تشارك في جريان مركز الوادي → "ظهرة"
```

#### **المحور الثالث: خصائص المياه والرطوبة (مفصل شامل)**

| النوع | الدلالة | المؤشرات | مصادر التحقق | الأهمية | درجة الثقة |
|--------|----------|-----------|-----------------|----------|-----------|
| **معين** | ينبع الماء من الأرض | عين مائية مرئية مباشرة | صور عالية الدقة، زيارة ميدانية | عالية جداً | 95% |
| **شط** | محاذية لمجاري السيول | بجانب وادي دائم الجريان | خريطة الجريان + صور | عالية | 90% |
| **خزجة** | تربة مشبعة بالماء | رطوبة عالية دائمة (NDVI > 0.7) | مؤشرات طيفية + تربة | عالية | 85% |
| **مسوح** | متأثرة بمياه الأمطار | تربة رطبة موسمياً | NDVI متغير موسمياً | متوسطة | 70% |
| **غارقة** | معرضة للغمر | تغرق عند الأمطار الغزيرة | في منطقة تجميع منخفضة | متوسطة | 65% |

**كيفية التحديد:**
```
من خرائط الجريان والمياه من الوكيل الرابع:

1️⃣ افحص وجود عين مائية:
   ├─ نعم → "معين" (ثقة 95%)
   └─ لا → انتقل للخطوة التالية

2️⃣ افحص الموقع بالنسبة للجريان:
   ├─ بجانب خط جريان دائم → "شط" (ثقة 90%)
   └─ لا → انتقل للخطوة التالية

3️⃣ افحص مؤشرات الرطوبة:
   ├─ من صور متعددة الأطياف:
   │  ├─ NDVI (مؤشر الغطاء النباتي)
   │  ├─ NDMI (مؤشر الرطوبة)
   │  └─ NDBI (مؤشر البناء)
   │
   ├─ إذا كان NDVI > 0.65 دائماً → "خزجة" (ثقة 85%)
   ├─ إذا كان NDVI متغير موسمياً → "مسوح" (ثقة 70%)
   └─ انتقل للخطوة التالية

4️⃣ افحص موقع التجميع المائي:
   ├─ في منطقة تجميع عالي في Slope Aspect
   ├─ أقل من الارتفاع المحيط
   └─ معرضة للغمر → "غارقة" (ثقة 65%)

5️⃣ اختر التصنيف الأفضل:
   └─ قد تطابق الأرض أكثر من فئة → اختر الأساسي
```

#### **المحور الرابع: نوع التربة والخصوبة (مفصل كامل)**

| النوع | الخصائص | المؤشرات من الصور | الطريقة | الموثوقية |
|--------|-----------|--------------------------|----------|-----------|
| **حجرات** | تربة مع حجارة صغيرة | رؤية واضحة للحجارة في الصور عالية الدقة | فحص بصري مباشر | عالية |
| **دقيقة** | تربة ناعمة جداً | نسيج ناعم جداً، عادة في وسط الوادي | تحليل ملمس الصورة | متوسطة |
| **مخلطة** | تربة + حجارة متوازنة | نسيج متوازن، حجارة ملحوظة لكن ليست الأغلبية | تحليل النسيج | متوسطة |
| **جوحية** | صخور مدفونة | تربة على السطح لكن صخور تحتها | معرفة جيولوجية + حفر تجريبية | منخفضة |
| **حمراء** | تربة حمراء اللون | لون أحمر واضح أو برتقالي | تحليل طيفي (Band Ratio) | عالية |
| **سهلة** | سهلة الحرث وخصبة | نسيج منتظم، لون داكن (غنية بالمادة العضوية) | تحليل طيفي + NDVI | عالية |

**كيفية التحديد:**
```
من تحليل صور فضائية وطيفية:

الخطوة 1: افحص الألوان الأساسية:
  ├─ إذا كانت حمراء/برتقالية واضحة → "حمراء"
  ├─ إذا كانت بنية غامقة → "سهلة" و "دقيقة"
  ├─ إذا كانت بنية فاتحة → "مخلطة"
  └─ إذا كانت رمادية مع نقاط بيضاء → "حجرات"

الخطوة 2: افحص النسيج والملمس:
  ├─ استخدم تحليل الملمس (Texture Analysis)
  ├─ نسيج ناعم جداً → "دقيقة"
  ├─ نسيج خشن → "حجرات"
  └─ نسيج متوسط → "مخلطة"

الخطوة 3: افحص مؤشرات الخصوبة:
  ├─ NDVI عالي (> 0.6) → تربة خصبة → "سهلة"
  ├─ NDVI منخفض (< 0.4) → تربة فقيرة → "حجرات"
  ├─ NDVI متوسط (0.4-0.6) → تربة متوسطة → "مخلطة"
  └─ حسب النتيجة، احدّد النوع

الخطوة 4: طبّق معادلة النسبة الطيفية:
  ├─ Red/Blue Ratio عالي → احتمال تربة حمراء
  ├─ Vegetation Index عالي → تربة خصبة
  └─ استخدم البرامج (QGIS, ArcGIS) لحسابها

الخطوة 5: احسب درجة الثقة:
  ├─ إذا كانت جميع المؤشرات تشير لنوع واحد → ثقة عالية (80-90%)
  ├─ إذا كانت المؤشرات متضاربة → ثقة منخفضة (40-60%)
  └─ في الحالات غير الواضحة → طلب مراجعة بشرية
```

### **الجزء الثاني: قواعد تصنيف المباني (مفصل شامل)**

#### **قواعد تصنيف المباني السكنية**

| المؤشر | القيمة 1 | القيمة 2 | القيمة 3 | القيمة 4 |
|--------|----------|----------|----------|----------|
| **المساحة** | < 50 م² | 50-100 م² | 100-300 م² | > 300 م² |
| **التصنيف** | سكني صغير جداً | سكني صغير | سكني متوسط | سكني فخم |
| **درجة الثقة** | عالية | عالية | عالية | عالية |
| **الاستخدام** | دور/ غرفة | بيت صغير | بيت عائلي | فيلا |

**معايير إضافية شاملة:**

#### **1. عمر البناء (من تاريخ الصورة):**

```
بيانات موثوقة:

جديد جداً (< سنة):
├─ ألوان زاهية جداً
├─ حدود حادة وواضحة جداً
├─ مواد البناء والحديد مرئية
├─ لا يوجد تأثر بالعوامل الجوية
└─ غالباً لم تكتمل الأعمال بعد

جديد (1-5 سنوات):
├─ ألوان زاهية
├─ حدود واضحة
├─ بدء ظهور علامات التقادم الطفيفة
├─ قد توجد بعض الشقوق الصغيرة
└─ المواد الأولية لا تزال مرئية

متوسط العمر (5-20 سنة):
├─ ألوان عادية
├─ حدود واضحة لكن ليست حادة
├─ علامات تقادم ملحوظة
├─ بدء تلون الجدران
├─ بعض الشقوق والتآكل
└─ الطراز معروف وشائع

قديم (20-40 سنة):
├─ ألوان بهتة
├─ حدود غير واضحة جداً
├─ تشقق واضح وتلون شديد
├─ قد توجد أجزاء مهدمة أو معاد بناؤها
├─ الطراز قديم وعتيق
└─ قد توجد تعديلات طفيفة

قديم جداً (> 40 سنة):
├─ ألوان جداً بهتة / تاريخية
├─ حدود غير واضحة جداً
├─ تشقق وتدمير واضح جداً
├─ أجزاء كبيرة قد تكون مهدمة
├─ مادة البناء قد لا تكون مرئية
└─ قيمة تاريخية محتملة
```

#### **2. نوع الاستخدام:**

```
سكني (Residential):
├─ المعايير:
│  ├─ نوافذ منتظمة على جميع الجهات
│  ├─ فناء أمامي أو حديقة
│  ├─ حجم متوسط (لعائلة واحدة)
│  ├─ مدخل واضح
│  └─ علامات سكن (نباتات، غسيل معلق)
│
└─ درجة الثقة: عالية جداً (90-95%)

تجاري (Commercial):
├─ المعايير:
│  ├─ واجهة كبيرة وواضحة (Storefront)
│  ├─ علامات تجارية أو نوافذ عرض
│  ├─ مدخل كبير وواضح
│  ├─ قد يكون على الطريق الرئيسي
│  └─ حجم متنوع (صغير لمتوسط)
│
└─ درجة الثقة: عالية (85-90%)

حكومي / عام (Government/Public):
├─ المعايير:
│  ├─ حجم كبير جداً (مبنى مؤسسة)
│  ├─ معمارية رسمية (symmetry, style)
│  ├─ علم أو شعار حكومي (في بعض الحالات)
│  ├─ موقع مركزي أو مهم
│  ├─ حديقة أو ساحة أمامية
│  └─ قد تكون محاطة بسور
│
└─ درجة الثقة: عالية (80-90%)

صناعي (Industrial):
├─ المعايير:
│  ├─ أسطح كبيرة وقاسية
│  ├─ هياكل معدنية أو خرسانية ضخمة
│  ├─ مداخل كبيرة لآلات / شاحنات
│  ├─ قد توجد أنابيب أو أسلاك خارجية
│  ├─ محاط بسور عادة
│  └─ موقع بعيد عن السكن عادة
│
└─ درجة الثقة: عالية جداً (90-95%)

زراعي (Agricultural):
├─ المعايير:
│  ├─ حجم صغير إلى متوسط
│  ├─ في وسط أراضٍ زراعية
│  ├─ قد يكون من الطوب اللبن أو الحجر
│  ├─ لا توجد تسهيلات حديثة
│  └─ قد يكون مهجوراً جزئياً
│
└─ درجة الثقة: متوسطة (65-75%)
```

#### **3. المادة الإنشائية:**

```
حجر طبيعي (Natural Stone):
├─ المؤشرات:
│  ├─ ألوان طبيعية متنوعة
│  ├─ سطح غير منتظم (rough texture)
│  ├─ ألوان تتراوح بين البني والرمادي
│  └─ عادة تاريخية / قديمة
│
└─ الاستخدام: عالي جداً (قيمة تاريخية)

حجر صناعي / خرسانة (Concrete/Artificial Stone):
├─ المؤشرات:
│  ├─ ألوان موحدة جداً
│  ├─ سطح منتظم (smooth)
│  ├─ عادة أبيض أو رمادي فاتح
│  └─ عادة حديثة نسبياً
│
└─ الاستخدام: عالي (متوسط العمر)

طوب أحمر (Red Brick):
├─ المؤشرات:
│  ├─ ألوان حمراء / برتقالية واضحة
│  ├─ نسيج خشن قليلاً
│  ├─ قد يكون محاط بملاط أبيض
│  └─ عادة عمر متوسط (20-30 سنة)
│
└─ الاستخدام: شائع جداً

خرسانة مسلحة (Reinforced Concrete):
├─ المؤشرات:
│  ├─ سطح رمادي فاتح موحد
│  ├─ أشكال هندسية دقيقة (rectangular)
│  ├─ قد تظهر الحديدات أحياناً
│  └─ عادة حديثة
│
└─ الاستخدام: عالي جداً (حديثة)

مواد خليطة (Mixed Materials):
├─ المؤشرات:
│  ├─ خليط من الألوان والملامس
│  ├─ قد يكون طوب + خرسانة
│  ├─ قد يكون حجر + طوب
│  └─ عادة إضافات وتعديلات
│
└─ الاستخدام: متوسط (يصعب التصنيف)
```

### **الجزء الثالث: قواعد تصنيف الطرق (مفصل شامل)**

#### **التصنيف الهرمي للطرق (تفصيل كامل)**

| المستوى | النوع | العرض | الخصائص | السرعة | الأهمية | الأمثلة |
|---------|--------|--------|-----------|--------|----------|----------|
| **1** | طريق دولي (محجة) | > 20 م | معبدة، ثنائية الاتجاه على الأقل، حارات واضحة | 100+ كم/س | دولية | الطريق السريع |
| **2** | طريق إقليمي | 15-20 م | معبدة، ثنائية الاتجاه، علامات وضوح | 80-100 كم/س | إقليمية | طريق بين محافظات |
| **3** | طريق سلطانية | 10-15 م | معبدة أو مرصوفة بشكل جيد | 60-80 كم/س | محلية كبرى | طريق بين مدن |
| **4** | طريق سبيل | 8-10 م | مرصوفة جزئياً | 40-60 كم/س | محلية | طريق بين بلدان |
| **5** | طريق مخلف | 5-8 م | غير معبدة لكن مستخدمة | 20-40 كم/س | محلية فرعية | طريق محلي |
| **6** | طريق ممر | < 5 م | مسار فقط، قد يكون تراب | < 20 كم/س | فرعية جداً | ممر في المنطقة |

**معايير القرار التفصيلية:**

```
الخطوة 1: احسب عرض الطريق
  ├─ استخدم صور عالية الدقة (< 5 متر بكسل)
  ├─ قس العرض من الحافة لحافة الأسفلت / الرصف
  ├─ خذ عدة قياسات (قد لا يكون موحد العرض)
  └─ استخدم المتوسط

الخطوة 2: حدّد حالة الرصف
  ├─ معبّدة (Asphalt / Concrete):
  │  ├─ سوداء وتوصل كهربائياً
  │  ├─ سطح أملس نسبياً
  │  └─ علامات طلاء بيضاء واضحة
  │
  ├─ مرصوفة (Paved / Graveled):
  │  ├─ سطح أفتح اللون من الأسفلت
  │  ├─ حصى / بحص على سطح
  │  └─ علامات بيضاء قد تكون غير واضحة
  │
  └─ ترابية (Unpaved / Dirt Road):
     ├─ بني فاتح / بني غامق
     ├─ سطح غير منتظم
     └─ لا توجد علامات

الخطوة 3: حدّد عدد الاتجاهات والحارات
  ├─ أحادي الاتجاه (One-way):
  │  ├─ سهم طريق واحد أو اتجاه واحد فقط
  │  └─ عادة طريق محلي صغير
  │
  ├─ ثنائي الاتجاه (Two-way):
  │  ├─ سهمان متعاكسان
  │  ├─ قد يكون خط أبيض في الوسط
  │  └─ عادة طريق إقليمي
  │
  └─ متعدد الحارات (Multi-lane):
     ├─ 4+ حارات (2+ لكل اتجاه)
     ├─ علامات فاصلة بيضاء واضحة
     └─ عادة طريق دولي/سريع

الخطوة 4: حدّد الكثافة المرورية (من صور متتالية)
  ├─ كثافة عالية:
  │  ├─ سيارات متتالية مرئية بوضوح
  │  ├─ بصمات إطارات واضحة جداً
  │  └─ حركة دائمة حتى الليل (جزئياً)
  │
  ├─ كثافة متوسطة:
  │  ├─ سيارات متباعدة
  │  ├─ حركة منتظمة في أوقات معينة
  │  └─ بصمات إطارات واضحة لكن ليست كثيفة جداً
  │
  └─ كثافة منخفضة:
     ├─ سيارات نادرة
     ├─ بصمات إطارات قد تكون غير واضحة
     └─ قد لا تكون مستخدمة يومياً

الخطوة 5: ابحث في الجدول أعلاه
  ├─ استخدم العرض كمعيار أساسي
  ├─ تحقق من حالة الرصف
  ├─ تحقق من عدد الاتجاهات
  └─ اختر المستوى الأنسب

الخطوة 6: احسب درجة الثقة
  ├─ إذا كانت جميع المؤشرات واضحة → ثقة عالية (85-95%)
  ├─ إذا كانت بعض المؤشرات غير واضحة → ثقة متوسطة (65-80%)
  └─ إذا كانت معظم المؤشرات غامضة → ثقة منخفضة (40-60%)
```

### **الجزء الرابع: دمج البيانات الخارجية (تفصيل شامل)**

**مصادر البيانات الخارجية المتاحة:**

```
1. geoBoundaries (الحدود الإدارية):
   ├─ الموقع: https://www.geoboundaries.org/
   ├─ البيانات المتاحة:
   │  ├─ حدود الدول
   │  ├─ حدود المحافظات / الولايات
   │  ├─ حدود المناطق الإدارية
   │  ├─ حدود المدن والبلدات
   │  └─ حدود المناطق الانتخابية
   │
   ├─ الفوائد:
   │  ├─ تحديد اسم المنطقة بدقة
   │  ├─ التحقق من الحدود المستخرجة
   │  ├─ توثيق رسمي معترف به
   │  └─ مرجع قانوني للتصرفات
   │
   └─ الاستخدام في الوكيل الثاني:
      ├─ مطابقة كل أرض مع منطقة إدارية
      ├─ التحقق من الأسماء الرسمية
      └─ حساب نسبة التوافق

2. ESA WorldCover (الغطاء الأرضي العالمي):
   ├─ الموقع: https://worldcover.org/
   ├─ البيانات المتاحة:
   │  ├─ تصنيف 10 فئات عالمية
   │  ├─ دقة 10 متر
   │  ├─ تغطية عالمية 100%
   │  └─ تحديثات سنوية
   │
   ├─ الفئات:
   │  ├─ 10: شجيرات وأشجار
   │  ├─ 20: أشجار
   │  ├─ 30: غطاء عشبي
   │  ├─ 40: محاصيل
   │  ├─ 50: بناء
   │  ├─ 60: مياه عارية
   │  ├─ 70: ثلج/جليد
   │  ├─ 80: سحب
   │  ├─ 90: ألواح شمسية
   │  └─ 95: مياه
   │
   └─ الاستخدام في الوكيل الثاني:
      ├─ التحقق من التصنيف الأولي
      ├─ الحصول على تصنيف مرجعي عالمي
      └─ حساب درجة الاتفاق

3. البيانات التاريخية والمحلية:
   ├─ الأسماء التقليدية والتاريخية:
   │  ├─ أسماء قديمة للمناطق
   │  ├─ أسماء الأودية والجبال
   │  ├─ أسماء المناطق السكنية
   │  └─ معايير تسمية محلية
   │
   ├─ الخرائط القديمة:
   │  ├─ خرائط عثمانية (إن وجدت)
   │  ├─ خرائط استعمارية
   │  ├─ خرائط الحقبة المبكرة
   │  └─ رسومات ورقية قديمة
   │
   ├─ المعلومات من السكان:
   │  ├─ أسماء محلية معروفة
   │  ├─ استخدامات تاريخية
   │  ├─ حدود أراضٍ معروفة
   │  └─ تاريخ الاستيطان
   │
   └─ الاستخدام في الوكيل الثاني:
      ├─ التحقق من أسماء الأماكن
      ├─ فهم الاستخدام التاريخي
      ├─ التقسيم التقليدي للأراضي
      └─ تعزيز درجة الثقة

4. نتائج الوكيل الرابع (الهيدرولوجيا):
   ├─ خرائط الجريان:
   │  ├─ اتجاه الجريان السطحي
   │  ├─ تجميع الجريان
   │  └─ حدود المستجمعات المائية
   │
   ├─ بيانات الانحدار:
   │  ├─ درجات الانحدار (Slope)
   │  ├─ اتجاه المنحدر (Aspect)
   │  └─ تحدب المنحدر (Curvature)
   │
   ├─ خرائط التضاريس:
   │  ├─ ارتفاع النقاط
   │  ├─ خطوط الكنتور
   │  └─ قمم الجبال والوديان
   │
   └─ الاستخدام في الوكيل الثاني:
      ├─ تحديد نوع الموقع (وادي/شعب/ظهرة)
      ├─ تحديد خصائص المياه
      ├─ فهم التصريف الطبيعي
      └─ التنبؤ بالرطوبة والسيول
```

**عملية الدمج الكاملة:**

```
لكل كيان مصنف من الوكيل الثاني:

الخطوة 1: اجلب التصنيف الأولي
  └─ من الوكيل الثاني: النوع المحدد + درجة الثقة

الخطوة 2: قارن مع geoBoundaries
  ├─ هل يقع في المنطقة الإدارية المتوقعة؟
  ├─ هل الحدود الإدارية توافق الحدود الطبيعية؟
  ├─ احسب درجة التوافق (0-100%)
  └─ وثّق أي تضارب

الخطوة 3: قارن مع ESA WorldCover
  ├─ هل التصنيف ESA يوافق التصنيف الأولي؟
  ├─ درجة التوافق (نسبة مئوية)
  ├─ تحقق من الفئات القريبة أيضاً
  └─ احسب نسبة اتفاق شاملة

الخطوة 4: تحقق من البيانات التاريخية:
  ├─ هل كان هذا الاستخدام موجود تاريخياً؟
  ├─ هل الاسم المحلي يطابق الكيان الحالي؟
  ├─ درجة الدعم من البيانات التاريخية
  └─ وثّق أي تغييرات ملحوظة

الخطوة 5: استخدم بيانات الهيدرولوجيا:
  ├─ هل الموقع يوافق خصائص الجريان المتوقعة؟
  ├─ هل مستوى الرطوبة يتطابق مع التصنيف؟
  ├─ هل خطر السيول مطابق للموقع؟
  └─ احسب درجة الاتفاق

الخطوة 6: احسب درجة الثقة النهائية:
  ├─ صيغة شاملة:
  │
  │  ثقة_نهائية = (ثقة_أولية × 0.4)
  │              + (توافق_geoBoundaries × 0.15)
  │              + (توافق_ESA × 0.15)
  │              + (دعم_تاريخي × 0.15)
  │              + (توافق_هيدرولوجي × 0.15)
  │
  └─ النتيجة: نسبة مئوية (0-100%)

الخطوة 7: اتخذ القرار النهائي:
  ├─ إذا ثقة > 80% → قبول وتوثيق
  ├─ إذا 60-80% → قبول مع ملاحظة
  ├─ إذا 40-60% → مراجعة بشرية مطلوبة
  └─ إذا < 40% → رفض كامل

الخطوة 8: وثّق القرار:
  ├─ النوع المختار النهائي
  ├─ درجة الثقة الكلية
  ├─ جميع درجات الثقة الفرعية
  ├─ أي تضاربات وكيف تم حلها
  └─ الملاحظات والتحفظات
```

## المخرجات النهائية للنظام

### **المجموعة 1: الخرائط (صيغ متعددة)**

```
1. خريطة الفئات الأساسية (5 فئات)
   ├─ صيغ: GeoJSON, SHP, GeoTIFF
   ├─ دقة: 2-5 متر
   └─ تغطية: 100% من منطقة الدراسة

2. خريطة التصنيف التفصيلي الشامل
   ├─ كل كيان برنه المحدد
   ├─ ألوان مختلفة لكل نوع (20+ لون)
   ├─ درجات ثقة موضحة
   └─ صيغ: GeoJSON, SHP, GeoTIFF

3. خريطة التغيرات عبر الزمن
   ├─ ما ظهر جديد (أزرق)
   ├─ ما اختفى قديم (أحمر)
   ├─ ما لم يتغير (رمادي)
   └─ خرائط منفصلة لكل فترة

4. خريطة الأخطار والتعريضات
```

## 📊 جداول مرجعية شاملة

### **جدول درجات الثقة الموحد (معايير الحكم)**

```
درجة الثقة    الوصف          الإجراء              الحالة
─────────────────────────────────────────────────────────────────
95-100%    عالية جداً    قبول فوري           ✅ موثق مباشرة
90-95%     عالية جداً    قبول + مراجعة سريعة  ✅ موثق + ملاحظة
80-90%     عالية         قبول + توثيق         ✅ موثق
70-80%     جيدة          قبول مع ملاحظة       ⚠️ توثيق مع تحفظ
60-70%     متوسطة        مراجعة بشرية مطلوبة  ⚠️ قيد المراجعة
50-60%     منخفضة        مراجعة بشرية إجبارية ❌ يحتاج تأكيد
< 50%      منخفضة جداً   رفض كامل              ❌ رفض
```

### **معادلات حساب درجات الثقة الشاملة**

#### **1. ثقة الأرض الزراعية (Formula الأساسية):**

```
ثقة_الأرض = (ثقة_الحجم × 0.35) 
          + (ثقة_الموقع × 0.25) 
          + (ثقة_المياه × 0.25) 
          + (ثقة_التربة × 0.15)

مثال:
├─ ثقة_الحجم = 85% (تطابق جيد مع معايير "كُرْوَة")
├─ ثقة_الموقع = 90% (موقع "وادي" واضح من DEM)
├─ ثقة_المياه = 80% (وجود شط مؤكد)
├─ ثقة_التربة = 75% (تربة مخلطة مع احتمال آخر)
│
└─ النتيجة:
   = (85 × 0.35) + (90 × 0.25) + (80 × 0.25) + (75 × 0.15)
   = 29.75 + 22.5 + 20 + 11.25
   = 83.5% (ثقة عالية ✅)
```

#### **2. ثقة المبنى:**

```
ثقة_المبنى = (ثقة_المساحة × 0.4) 
           + (ثقة_العمر × 0.3) 
           + (ثقة_الاستخدام × 0.2)
           + (ثقة_المادة × 0.1)

مثال:
├─ ثقة_المساحة = 90% (واضح جداً)
├─ ثقة_العمر = 75% (بعض الغموض)
├─ ثقة_الاستخدام = 85% (سكني واضح)
└─ ثقة_المادة = 80% (حجر طبيعي)

└─ النتيجة = (90×0.4) + (75×0.3) + (85×0.2) + (80×0.1)
           = 36 + 22.5 + 17 + 8
           = 83.5% (ثقة عالية ✅)
```

#### **3. ثقة الطريق:**

```
ثقة_الطريق = (ثقة_العرض × 0.4) 
           + (ثقة_الرصف × 0.35) 
           + (ثقة_الاتجاهات × 0.15)
           + (ثقة_الكثافة × 0.1)
```

#### **4. الثقة النهائية الشاملة (بعد دمج البيانات):**

```
ثقة_نهائية = (ثقة_أولية × 0.40)
           + (توافق_geoBoundaries × 0.15)
           + (توافق_ESA × 0.15)
           + (دعم_تاريخي × 0.15)
           + (توافق_هيدرولوجي × 0.15)

حيث:
├─ ثقة_أولية: من الوكيل الثاني
├─ توافق_geoBoundaries: 0-100% حسب التطابق
├─ توافق_ESA: 0-100% حسب التصنيف المطابق
├─ دعم_تاريخي: 0-100% حسب التأكيد التاريخي
└─ توافق_هيدرولوجي: 0-100% حسب توافق DEM والجريان
```

#### **5. حساب نسبة التطابق (Matching Percentage):**

```
نسبة_التطابق = (عدد_المعايير_المتطابقة / إجمالي_المعايير) × 100%

أو بصيغة مرجحة:
نسبة_التطابق = Σ(معامل_الوزن × درجة_التطابق) / Σ(معاملات_الأوزان)

مثال:
├─ معيار المساحة: 90% توافق × وزن 0.3 = 27%
├─ معيار العرض: 85% توافق × وزن 0.3 = 25.5%
├─ معيار الطول: 70% توافق × وزن 0.2 = 14%
└─ معيار الموقع: 95% توافق × وزن 0.2 = 19%

└─ النتيجة = 27 + 25.5 + 14 + 19 = 85.5% توافق
```

## 🚀 خطة الاختبار والتطبيق المفصلة

### **المرحلة 1: الاختبار على عينة صغيرة (الأسابيع 1-2)**

```
المحطة الأولى: اختيار وتحضير العينة

1️⃣ اختر منطقة جغرافية محدودة:
   ├─ الحجم: 5×5 كم (25 كم²) تقريباً
   ├─ الخصائص: متنوعة (أراضي، مباني، طرق، مياه)
   ├─ المتطلبات:
   │  ├─ صور عالية الدقة (< 5 متر)
   │  ├─ بيانات DEM متاحة
   │  ├─ بيانات تاريخية للمقارنة
   │  └─ إمكانية زيارة ميدانية
   └─ المثال: مدينة صغيرة أو منطقة ريفية محدودة

2️⃣ جمع البيانات المرجعية (Ground Truth):
   ├─ صور جوية عالية الدقة جداً (< 1 متر)
   ├─ مسح ميداني دقيق (GPS، تحديد يدوي)
   ├─ الأسماء المحلية والتاريخية
   ├─ خريطة ورقية قديمة إن وجدت
   └─ شهادات من السكان المحليين

3️⃣ تصنيف يدوي كمرجع (Reference Classification):
   ├─ خبير يصنف الكيانات يدوياً
   ├─ توثيق كامل للتصنيفات
   ├─ صور توضيحية لكل نوع
   └─ درجة ثقة عالية جداً (95%+)

4️⃣ إنشاء قاعدة البيانات المرجعية:
   ├─ 200-300 كيان على الأقل
   ├─ توزيع متساوي للأنواع المختلفة
   ├─ توثيق كامل للخصائص
   └─ احتفظ بـ 80% للتدريب و 20% للاختبار

───────────────────────────────────────────────────────

المحطة الثانية: تشغيل النظام الكامل

5️⃣ شغّل الوكيل الأول (Coarse Classification):
   ├─ استخراج الفئات الأساسية
   ├─ فحص النتائج بصرياً
   ├─ تسجيل أي أخطاء واضحة
   └─ حفظ المخرجات

6️⃣ شغّل الوكيل الرابع (Hydrological Analysis):
   ├─ استخراج DEM والمعالجة
   ├─ تحليل التضاريس والجريان
   ├─ التحقق من جودة النتائج
   └─ حفظ جميع الطبقات

7️⃣ شغّل الوكيل الثاني (Detailed Classification):
   ├─ تطبيق جميع القواعس
   ├─ دمج البيانات الخارجية
   ├─ حساب درجات الثقة
   └─ إنتاج الخريطة التفصيلية

8️⃣ شغّل الوكيل الثالث (Analysis and Decision Support):
   ├─ إكمال الإرجاع الجغرافي
   ├─ تحليل الأنماط والتغيرات
   ├─ إعداد التقرير
   └─ عرض على خبير للتصديق

───────────────────────────────────────────────────────

المحطة الثالثة: تقييم الأداء

9️⃣ احسب المقاييس الإحصائية:
   
   أ) مقاييس الدقة:
      ├─ Accuracy = (TP + TN) / (TP + TN + FP + FN)
      ├─ Precision = TP / (TP + FP)
      ├─ Recall = TP / (TP + FN)
      ├─ F1-Score = 2 × (Precision × Recall) / (Precision + Recall)
      └─ Kappa = (Accuracy_Observed - Accuracy_Expected) / (1 - Accuracy_Expected)
      
      حيث:
      ├─ TP (True Positive): صنفنا صح + الحقيقة صح
      ├─ TN (True Negative): صنفنا خطأ + الحقيقة خطأ
      ├─ FP (False Positive): صنفنا صح + الحقيقة خطأ
      └─ FN (False Negative): صنفنا خطأ + الحقيقة صح
   
   ب) مقاييس الأخطاء:
      ├─ Mean Absolute Error (MAE)
      ├─ Root Mean Square Error (RMSE)
      ├─ Positional Accuracy
      └─ Thematic Accuracy

🔟 قارن النتائج مع المرجع:
   ├─ مقارنة نوع بنوع
   ├─ تحليل الأخطاء الشائعة
   ├─ تحديد الأسباب الجذرية
   └─ توثيق الملاحظات

───────────────────────────────────────────────────────

المحطة الرابعة: التحسين والتعديل

1️⃣1️⃣ حلل نتائج الاختبار:
   ├─ أي النقاط أداؤها جيد؟ (أكثر من 85%)
   ├─ أي النقاط أداؤها ضعيف؟ (أقل من 70%)
   ├─ ما الأنماط في الأخطاء؟
   └─ هل هناك تحيزات معينة؟

1️⃣2️⃣ اضبط القواعس إذا لزم:
   ├─ هل المعايير الرقمية صحيحة؟
   ├─ هل أوزان الخصائص موزونة بشكل صحيح؟
   ├─ هل هناك معايير ناقصة؟
   └─ هل هناك معايير زائدة غير ضرورية؟

1️⃣3️⃣ حسّن البيانات الخارجية:
```

## 📝 الملاحظات والتوصيات النهائية

### **ملاحظات مهمة جداً:**

```
1️⃣ الدقة مقابل السرعة:
   ├─ النظام يركز على الدقة أكثر من السرعة
   ├─ قد يستغرق وقت أطول في البداية
   ├─ لكن النتائج ستكون موثوقة جداً
   └─ كل قيمة = الدقة تستحق الانتظار

2️⃣ المراجع البشري ضروري:
   ├─ لا يمكن للنظام أن يكون 100% دقيق
   ├─ الحالات الملتبسة تحتاج تحكيم بشري
   ├─ الخبرة المحلية غير قابلة للحلول الآلية
   └─ استثمر في تدريب المراجعين

3️⃣ البيانات الجيدة = نتائج جيدة:
   ├─ "Garbage in, garbage out" (قمامة في = قمامة خارج)
   ├─ جودة الصور الأولية حاسمة
   ├─ البيانات التاريخية يجب أن تكون موثوقة
   └─ اجمع أفضل البيانات المتاحة

4️⃣ التوثيق شامل وضروري:
   ├─ كل قرار يجب أن يكون موثق
   ├─ كل حالة شاذة يجب أن توثق
   ├─ كل تغيير في القواعس يجب أن يسجل
   └─ الملفات والإصدارات يجب أن تكون منظمة

5️⃣ النظام يجب أن يكون مرنا:
   ├─ القواعس قابلة للتعديل حسب المنطقة
```

### **ملحق 3: معادلات رياضية متقدمة**

#### **مؤشرات طيفية مستخدمة:**

```
NDVI (Normalized Difference Vegetation Index):
NDVI = (NIR - RED) / (NIR + RED)
حيث: -1 ≤ NDVI ≤ 1
├─ < 0: مياه
├─ 0-0.3: أرض حجرية / بناء
├─ 0.3-0.6: نباتات قليلة / عشب
└─ > 0.6: نباتات كثيفة / غابات

NDMI (Normalized Difference Moisture Index):
NDMI = (NIR - SWIR) / (NIR + SWIR)
├─ يقيس محتوى الرطوبة في التربة والنبات
└─ يستخدم لتصنيف رطوبة الأراضي

NDBI (Normalized Difference Built-up Index):
NDBI = (SWIR - NIR) / (SWIR + NIR)
├─ يقيس كثافة البناء
└─ يستخدم لاستخراج المباني والطرق

NDWI (Normalized Difference Water Index):
NDWI = (GREEN - NIR) / (GREEN + NIR)
├─ يقيس محتوى الماء السطحي
└─ يستخدم للمسطحات المائية

Brightness Index:
Brightness = √(R² + G² + B²) / 3
├─ يقيس السطوع الكلي
└─ يستخدم لتمييز الأجسام الفاتحة
```
