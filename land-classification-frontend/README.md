# وكيل تصنيف الأراضي - Frontend

نظام واجهة أمامية لتحليل وتصنيف المساحات الجغرافية باستخدام الذكاء الاصطناعي.

## مبدأ العمل

هذا المشروع يتبع فلسفة العمل التالية:

1. **استغلال ما هو جاهز** - استخدم نماذج ذكاء اصطناعي جاهزة، مكتبات مجانية، وخدمات سحابية بطبقة مجانية
2. **السرعة فوق الكمال** - حل يعمل اليوم بدقة 85% أفضل من حل يبنى من الصفر في 3 أشهر
3. **استخدم الذكاء الاصطناعي في برمجتك** - استخدم GPT, Claude, Copilot لكتابة الكود وتصحيح الأخطاء

## كيفية التشغيل

```bash
# تثبيت التبعيات
npm install

# تشغيل الخادم المحلي
npm run dev
```

افتح [http://localhost:3000](http://localhost:3000) لرؤية التطبيق.

## المميزات

### 1. الصفحة الرئيسية
- رفع ملفات Shapefile, GeoJSON, GeoTIFF
- اختيار منطقة الدراسة
- زر "ابدأ التصنيف"

### 2. صفحة النتائج
- عرض خريطة التصنيف مع الطبقات المختلفة
- تدرج ألوان للطبقات (غطاء نباتي، مسطحات مائية، عمراني، أرض عارية، أرض زراعية)
- عرض إحصائيات مبدئية
- زر "التدقيق" للانتقال لواجهة التدقيق

### 3. واجهة التدقيق
- عرض الخريطة مع النتائج الحالية
- أدوات التدقيق (تغيير التصنيف، إضافة ملاحظات)
- حفظ التعديلات وإرسالها للباكند
